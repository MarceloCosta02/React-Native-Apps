-- BAIXAR O YARN
-- BAIXAR O CHOCOLATEY

Para executar app backend:

node src/server.js ou yarn dev

Instalados no terminal do vs code

yarn add express -- instala a biblioteca express

yarn add nodemon -D -- instala o nodemon que faz com que você não precise reiniciar o server quando fizer alterações, só salvar e dar f5

yarn add mongoose -- ele é um odm, ferramenta que ajuda a trabalhar com banco de dados utilizando somente javascript

yarn add axios -- pacote para fazer requisições em apis externas

yarn add cors -- permitir que minha aplicação seja acessada de qualquer endereço

--------------------------------------------------------------------

para executar o app frontend

yarn start

yarn add react-router-dom -- versão para web da biblioteca que facilita o uso de rotas


yarn add axios -- instalar axios no frontend também

-------------------------------------------------------------------
npx react-native init (nome do projeto)

mobile - executar primeira vez > no cmd/powershell como adm: yarn react-native run-android

depois> yarn start na pasta do projeto mobile

instalar os 3 módulos no terminal:

yarn add react-navigation react-native-gesture-handler react-native-reanimated 
// o primeiro faz a parte de navegação básica, o segundo é uma dependencia do navigation, para lidar com gestos do usuario, e a terceira para animações nas transições de rota

acessar https://kmagiera.github.io/react-native-gesture-handler/docs/getting-started.html

e colocar o codigo em roxo no projeto em android/app/src/main/java/MainActivity.java

para manter o usuario logado ao atualizar a pag, ter um local storage, armazenar infos rapidas
yarn add @react-native-community/async-storage 

se der erro usar esse>> e depois yarn react-native run-android
